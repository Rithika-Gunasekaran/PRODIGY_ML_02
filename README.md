# Customer Segmentation using K-Means

## 📌 Project Description
This project is used to group customers based on their income and spending behavior using K-Means clustering algorithm.

## 📊 Dataset
Mall Customers dataset (CSV file)

## ⚙️ Steps Performed
1. Imported required libraries
2. Loaded dataset using pandas
3. Selected important features:
   - Annual Income
   - Spending Score
4. Used Elbow Method to find optimal number of clusters
5. Applied K-Means algorithm
6. Visualized customer groups using graph
7. Saved output as CSV file

## 📈 Output
- Elbow Method graph
- Customer Segmentation graph
- Output file with cluster labels (output.csv)

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## 🎯 Result
Customers are divided into 5 clusters based on their spending and income patterns.
